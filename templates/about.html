About Us

This project has been developed for the BCA 5th sem MPR (Minor Project Report).In this project, we are working on a diabetes prediction model for better classification of diabetes which includes few external factors responsible for diabetes along with regular factors like Glucose, BMI, Age, Insulin, etc. Classification accuracy is boosted with new dataset compared to existing dataset.
